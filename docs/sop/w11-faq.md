Q1：W11 4G版本和WiFi版本区别：

A： 两者的主要区别在于网络连接方式和使用场景：

- WiFi版：需连接2.4GHz WiFi网络，适合家庭内部有稳定WiFi覆盖的场景。日常用不消耗SIM卡流量。
    
- 4G版：内置4G模块和SIM卡（已预装），无需WiFi即可独立联网，适合家庭WiFi信号弱、或无宽带网络的户外/偏远场景使用。
    

除此之外，两款产品的核心功能（如黑光全彩、移动侦测、双向语音、云存储等）保持一致。

---

Q2：4G版本是否支持更换SIM卡？

A：不支持。用户无需也无法自行插拔或替换。

---

Q3：4G版需要自己购买流量套餐吗？  
A： 需要。用户需在首次使用前，通过HomiQ life APP购买数据套餐。具体资费请参考产品App内指引。

---

Q4：4G版的流量消耗大吗？每月需要多少流量？  
A： 流量消耗取决于使用强度：

- 日常待机：几乎不消耗流量。
    
- 频繁预览：消耗较多流量。
    

---

Q5：4G版摄像机在没有Wi-Fi的地方能正常使用吗？

A：可以。 4G版内置蜂窝网络模块，只要有当地4G网络覆盖并且用户订阅了流量包后即可正常使用。

---

Q6：4G版的网络信号强度如何？如果信号不好怎么办？

A：摄像机的信号强度取决于安装位置当地4G网络的覆盖质量。如果发现信号弱或连接不稳定，可按以下步骤优化：

1. 检查App内信号强度：登录配对摄像机的App，在预览页面查看4G信号格数。若只有1-2格，说明信号偏弱。
    
2. 调整安装位置：尝试将摄像机移动到靠近窗户、室外或更高处，避开厚墙、金属物、地下室等信号屏蔽严重的区域。
    
3. 重启设备：双击摄像机的Reset键关机1分钟，然后重启摄像机，有时能重新注册到信号更好的基站。

Q7：摄像机电池容量多大？能否拆卸？充满电需要多久？能工作多长时间？

A： 内置2000mAh可充电聚合物电池，禁止私自拆卸，以免损坏设备或造成安全隐患。

- 完全充满电需要约4-5小时。实际续航会因唤醒频率、报警触发次数、环境温度等因素有所差异。频繁预览或低温环境会缩短续航时间。这里提供一组参考数据：常温环境下，按每天唤醒 20 次、单次充满电可续航约 2个月。
    

---

Q8：如何延长电池续航时间？

A： 建议：

1. 降低侦测灵敏度。
    
2. 避免频繁在App中预览画面。
    
3. 关闭非必要的报警推送功能。

Q9：安装太阳能板时需要注意什么？

A： 为获得最佳充电效果，请注意：

1. 选择阳光充足的位置：日均直射光照建议不低于4小时。
    
2. 避免遮挡：远离树木、屋檐、空调外机等可能形成阴影的物体。
    
3. 高度适中：安装在便于清洁的位置，方便定期擦拭表面灰尘。
    
4. 距离适中：太阳能板附带约18厘米（0.6ft）连接线。摄像机与太阳能板不超过线缆长度即可。
    
5. 右侧安装：通常，太阳能板安装在摄像机的右侧。

Q11：太阳能板需要充多久才能把摄像机电池充满？

A： 充满时间取决于光照条件。在充足直射阳光（日均有效光照≥4小时） 下，约2-3个晴天可将耗尽的电池充满。阴雨天或冬季光照不足时，充电时间会明显延长。

注意：太阳能板主要作用是维持日常电量平衡，而非快速充电。在正常使用频率下（日均触发20次左右），只要光照充足，太阳能板足以抵消日常耗电，保持电池不亏电。

Q12：太阳能板无法充电，怎么办？

A： 可按以下顺序排查：

|   |   |
|---|---|
|检查连接|确认USB-C接口已插紧，无松动或氧化。|
|2.检查光照角度|让太阳能板尽量正对阳光。|
|3.检查遮挡|确保太阳能板周围无树枝、屋檐等阴影遮挡。|
|4.检查天气|连续阴雨天时充电效率下降属正常现象。建议在此类天气时使用适配器充电，适当降低灵敏度和减少预览来延长电池使用时间。|
|5.检查清洁度|1. 确保太阳能板表面无灰尘、定期清洁可提升充电效率。|

Q13：长期阴雨天后电池耗尽了，天气转晴用太阳能板充电，为什么摄像机还是无法开机？

A： 这是电池深度放电后的正常现象，并非设备故障。

原因说明：

- 电池长期亏电后电压会降至过放保护阈值以下，此时电池保护板会切断输出，防止电池损坏。
    
- 受到光照条件影响，太阳能板输出的电流小，无法直接“唤醒”处于深度休眠的电池保护板。
    
- 需要先用稍大电流激活电池，才能恢复正常充电。
    

解决方案：

|   |   |   |
|---|---|---|
|方法|操作说明|适用场景|
|1. 使用充电宝或5V充电器|断开太阳能板，用充电宝或5V/2A充电器直接给摄像机充电15-30分钟，电池被激活后再换回太阳能板|最快速有效|
|2. 拔插USB线|确认在晴天连续充电情况下，将连接太阳能板的USB-C线拔下，等待10秒后重新插紧。瞬时电流变化有时可触发保护板恢复|快速尝试，多数情况有效|

预防措施：

1. 设置低电量报警，一旦收到推送，及时充电。settings page -- Power Management Settings -- Select the level.
    
2. 冬季或长期阴雨天前，可在App中降低侦测灵敏度，减少预览来减少耗电。
    
3. 若连续阴雨天超过1周，建议提前用充电宝给摄像机补电一次，避免深度亏电。

Q14：摄像机是如何工作的？

A：摄像机采用智能休眠唤醒机制，并非24小时连续工作。

|   |   |   |
|---|---|---|
|状态|说明|优点|
|1. 休眠状态（无报警）|摄像机进入低功耗休眠模式，几乎不消耗电量。此时无法实时预览，但会保持网络连接等待唤醒指令。|延长了电池续航时间。在日均触发20-30次报警的典型使用场景下，单次充电可续航数周至数月。|
|2. 唤醒状态（侦测到报警）|当雷达传感器侦测到人体移动后，摄像机被唤醒，并推送报警通知到App。|
|3. 实时预览|用户打开App点击预览画面时，摄像机也会被唤醒，建立实时视频流。|

Q15：摄像机支持录像吗？

A：支持三种种录像选项。

1. 卡录像：摄像机支持一个Micro SD 卡槽，用户可插入256G以内的C10存储卡（64G以上格式化为exfat，以下格式化为FAT）。当有事件触发时会开始录像直到事件停止。
    
2. 云存储录像：当有事件触发时会开始录像直到事件停止。该功能需订阅。
    
3. 即时录像：在实时预览时点击页面录像图标进行录像，文件保存在相册中，可保存到手机本地相册。一旦结束预览录像就会停止。

Q16：摄像机支持哪些侦测功能？有什么区别？

A： 目前支持移动侦测和人形侦测两种功能，区别如下：

|   |   |   |
|---|---|---|
|功能|说明|推送条件|
|移动侦测（PIR）|侦测画面中任何物体的移动（包括人、动物、车辆、飘动的树叶等）。|只要有物体移动，就会推送报警|
|人形侦测|在移动侦测基础上，通过AI算法识别人形轮廓，过滤非人形警报。|仅当侦测到人时才推送报警，减少误报|

配置技巧：

1. 如果摄像机安装在人来人往的区域，建议同时开启人形侦测，可大幅减少无效报警。
    
2. 安装在需要监控所有活动的区域（如车库、仓库），仅开启PIR侦测即可。
