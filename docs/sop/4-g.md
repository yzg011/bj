# Operation guide

1. Place the jd_factory.ini file into the TF card, and create a folder named jd_laohua within the TF card.
![image.png](https://img.z2m.store/file/1779326278308_image.png)

2. After testing for a period of time, please send us the entire jd_laohua folder.

3. Remove the SIM card from the camera

4. Insert the SIM card into the phone

5. To check the APN settings for this card, use an Android phone. Note that this option is not available in newer versions of iOS, so do not use an iPhone to check it. And send us the screenshots you obtained.
![image.png](https://img.z2m.store/file/1779326285450_image.png)
