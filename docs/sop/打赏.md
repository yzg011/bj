# vitepress-plugin-sponsor

[](https://github.com/SlovinG/vitepress-plugin-sponsor#vitepress-plugin-sponsor)

美观的 vitepress 打赏插件，修改自 [donate-page](https://github.com/Kaiyuan/donate-page)

主要作用是在文章底部添加打赏模块：

[![alt text](https://github.com/SlovinG/vitepress-plugin-sponsor/raw/main/image.png)](https://github.com/SlovinG/vitepress-plugin-sponsor/blob/main/image.png)

## 使用

[](https://github.com/SlovinG/vitepress-plugin-sponsor#%E4%BD%BF%E7%94%A8)

安装依赖 `pnpm/npm/yarn`

```shell
pnpm add vitepress-plugin-sponsor
```

引入插件在 `.vitepress/config.ts` 配置文件中

```ts
import { defineConfig } from 'vitepress'
import { templatePlugin } from 'vitepress-plugin-sponsor'

export default defineConfig({
  vite: {
     plugins:[
        // 打赏插件
        SponsorPlugin({
            /**
             * 打赏模块样式
             */
            type: 'simple',
            aliPayQR: 'your_image_link',
            weChatQR: 'your_image_link'
        })
     ]
  }
})
```

## 完整配置

[](https://github.com/SlovinG/vitepress-plugin-sponsor#%E5%AE%8C%E6%95%B4%E9%85%8D%E7%BD%AE)

```ts
export interface SponsorOptions {
    /**
     * 打赏模块样式
     */
    type: 'simple',
    aliPayQR: '',
    weChatQR: '',
}
```
