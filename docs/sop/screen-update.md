# B2屏幕固件升级步骤
1. 安装升级工具：
![image.png](https://img.z2m.store/file/1780570728501_image.png)
2. 打开升级工具，打开升级工具后界面如下：
![image.png](https://img.z2m.store/file/1780570777008_image.png)
3. 点击“一键刷机”，点击界面中的浏览，选中要升级的固件（选中固件完成后电脑上不用点击其他操作）：
![image.png](https://img.z2m.store/file/1780570868213_image.png)
4. 使用数据线将屏幕和电脑连接

5. 按住放大键(或方向下键)，再点一下电源键，左侧指示灯会全部亮起，灯亮了之后可以松开放大键，迅速的连点电源键五次进入升级模式（执行操作之前，屏幕需要先关机）
![image.png](https://img.z2m.store/file/1780570926990_image.png)![image.png](https://img.z2m.store/file/1780570962480_image.png)
6. 等待几十秒之后，电脑显示“固件烧写成功”即可
![image.png](https://img.z2m.store/file/1780570995896_image.png)
