### 💡 建议操作流

对于 Astro 项目首次上传，标准流程如下：

```
# 1. 确保所有文件被添加
git add .

# 2. 提交变更（如果有变更）
git commit -m "Initial commit"

# 3. 重命名分支为 main（与 GitHub 默认一致）
git branch -M main

# 4. 推送到远程
git push -u origin main
```
