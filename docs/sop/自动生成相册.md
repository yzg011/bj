# 自动生成相册文件

## 使用方法

- **新建相册文件**: 在博客根目录新建相册文件夹blog\source\album
- **新建相册文件夹名称**: 如旅行/工作等
- **在对应的文件夹下建相册简介文件**: 新建desc.txt文件
第一行写相册名称
第二行写相册简介
- **保存相片**: 把相片保存在相应的文件夹
- **生成博客代码**: 执行命令自动生成博客识别的相册代码

## 快速开始

执行下列代码自动生成

```
node pic.js
```
下次增加完相片，执行下上述代码就可以啦！

看看我的相册就知道啦
